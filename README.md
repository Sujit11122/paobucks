
Paobucks is an Android coffee ordering app built with Kotlin. It allows users to browse coffee items, customize their orders, select add-ons and sizes, and calculate the total price before placing an order.

Features:

1. Browse a list of coffee items with descriptions and images.
   
2. Select coffee size (Small, Medium, Large) with dynamic pricing.

3. Choose add-ons like extra syrup, whipped cream, etc.

4. View total price based on selected options.

5. User-friendly UI with modern design patterns.

Technology Stack:

1. Language: Kotlin

2. Platform: Android Studio

3. UI Components: RecyclerView, CardView, Buttons, ImageView, TextView

4. Architecture: Basic Kotlin OOP principles
